**会社名**：[AnyMind Japan株式会社](https://anymindgroup.com/ja/)｜業務委託（フルリモート）  
**期間**：2022-04-05〜2022-04-17（12日間）  
**担当フェーズ**：要件定義（見積もり）  
**使用ツール**：Googleスプレッドシート／Google Meet  

#### 🔧 担当業務と工夫・成果
- ヘルスケア系iOS/Androidアプリの初期開発における機能一覧・画面構成をもとに、工数見積もり資料を作成
